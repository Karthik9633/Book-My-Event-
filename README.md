# 🚀 BookMyEvent  
### 🎟 Discover • Book • Experience  

A full-stack event discovery and booking platform with AI assistant, smart filtering, interactive maps, calendar view, authentication, ticket management, and premium UI experience.

🔗 Live Demo:https://book-my-event-ten.vercel.app/

---

## 🌟 Live Experience

**BookMyEvent** is a modern, production-style event platform built with React, featuring:

- 🔍 Smart search & live filters  
- 🗺 Google Maps integration  
- 📅 Interactive calendar  
- ❤️ Favorites system  
- 🎟 Dynamic ticket booking  
- 🔐 Authentication with protected routes  
- 🤖 AI Assistant (Uber-style chatbot UI)  
- 📱 Fully responsive design  
- 🎨 Premium UI/UX
